# ValorantIQ

A Valorant performance tracker that is able to analyze your match history, gives personalized coaching tips, and lets you compare players side by side.

**Live demo:** https://valorantiq-78xz7yqrabkv7enhzci8p2.streamlit.app

---

## Features

- Match stats — K/D ratio, headshot percentage, damage, assists
- ValorantIQ Score — a performance grade out of 100 based on your stats
- Personalized coaching tips based on your recent games
- Player comparison — go head to head with any Valorant player
- Agent and map breakdowns
- MMR and peak rank display
- Multi-region support (AP, NA, EU, KR)

---

## Running Locally

Clone the repo and install dependencies:

```bash
git clone https://github.com/Yoomin566/ValorantIQ.git
cd ValorantIQ
pip install -r requirements.txt
```

Create a `.env` file with your API keys:

RIOT_API_KEY=your-riot-key
HENRIK_API_KEY=your-henrik-key

**4. Run the app**
```bash
streamlit run Home.py
```
